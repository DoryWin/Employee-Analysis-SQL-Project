This project demonstrates foundational SQL skills through the creation and analysis of an Employee Management Database.
It includes table creation, data insertion, and a set of analytical and transformation queries focusing on salary, text operations, and department-based insights.

Dataset Summary

The dataset includes 20 employees from 3 departments:
IT (101) – Manager, Developer, Tester
HR (102) – Analyst, Recruiter, Manager
Finance (103) – Accountant, Auditor, Analyst
It covers employees hired between 2022–2023, with detailed salary information and job designations.

Analysis Questions & SQL Features Demonstrated

This project includes 12 SQL queries that showcase the use of:
String functions → LEFT, RIGHT, CONCAT, LOWER, UPPER, LEN
Date functions → YEAR()
Math operations → raises, bonuses, casting
Conditional filtering → WHERE statements
Data type conversions → CAST

The queries include:
Average salary of employees who joined in 2022
Extract and concatenate characters from name/phone
Convert designations to lowercase for IT employees
Count characters in address for high-salary employees
Calculate 5% salary increase
Calculate 8% salary bonus
Overtime pay at 15%
Double salary for IT employees
Concatenate employee info with dashes
Uppercase designation + lowercase department
Convert salary to integer
Add "Emp" prefix to employee numbers
